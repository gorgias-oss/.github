# Gorgias OSS

Open source tooling from [Gorgias](https://www.gorgias.com) — the customer support platform built for ecommerce.

We share the infrastructure tools, Terraform providers, and developer utilities we build internally so the broader engineering community can use and improve them.

---

## Projects

### Terraform Providers

| Provider | Description |
|---|---|
| [terraform-provider-altinity](https://github.com/gorgias-oss/terraform-provider-altinity) | Manage Altinity.Cloud environments and ClickHouse clusters via the ACM REST API |
| [terraform-provider-huggingface](https://github.com/gorgias-oss/terraform-provider-huggingface) | Manage Hugging Face Inference Endpoints — full lifecycle IaC for ML model deployments |

---

## Contributing

All repositories welcome issues and pull requests. Each project has a `CONTRIBUTING.md` with local dev setup, coding conventions, and the PR process.

For security disclosures, reach out to **security@gorgias.com**.
